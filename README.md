
# ربات پیشرفته مدیریت و سرگرمی گروه تلگرام

ربات تلگرامی قدرتمند با قابلیت‌های مدیریت گروه، سیستم سطح و تجربه، بازی‌ها، سرگرمی‌های متنوع، هوش مصنوعی و ده‌ها ویژگی دیگر.

## ویژگی‌ها

- مدیریت پیشرفته گروه (قفل، ضدلینک، ضدتبلیغ، ضدفحش، ضد هنگی، ضد ایموجی، ضد ویرایش، ضد منشن، ضد گیف)
- سیستم سطح و تجربه (XP) و اهدای نشان‌های ویژه
- سیستم اخطار (Warn) با جریمه خودکار (سکوت/اخراج/بن)
- پیام خوش‌آمدگویی و خداحافظی شخصی‌سازی شده
- سیستم کپچا برای جلوگیری از ربات‌ها
- دستورات سفارشی برای هر گروه
- بازی‌ها: ریاضی، کلمات، حدس عدد
- سرگرمی‌ها: فال حافظ، فال روز، لطیفه، حقیقت جالب، جمله فلسفی، معما، داستان، خاطره، حدیث، چالش روزانه
- تست شخصیت و طالع‌بینی
- محاسبه سن شمسی
- هوش مصنوعی (پاسخ به سوالات با `+سوال`)
- قیمت ارز دیجیتال و نرخ ارز
- تاریخ و زمان شمسی
- جستجو در مایکت
- تبدیل فونت
- تحلیل احساسات روزانه

## نصب و راه‌اندازی

1. کلون کنید:
```bash
git clone https://github.com/yourusername/bot.git
cd bot
```

2. نصب وابستگی‌ها:
```bash
pip install -r requirements.txt
```

3. توکن ربات خود را در خط زیر وارد کنید:
```python
bot = Robot("TOKEN_HERE", enable_offset=True, max_msg_age=90000)
```

4. اجرا:
```bash
python bot.py
```

## دستورات سریع

| دستور | توضیح |
|-------|-------|
| `/start` | شروع و آمار |
| `راهنما` | راهنمای کامل |
| `سطح` | نمایش سطح و تجربه |
| `نشان‌ها` | نشان‌های شما |
| `لیست برترین‌ها` | برترین کاربران گروه |
| `بازی ریاضی` | شروع بازی ریاضی |
| `فال حافظ` | فال حافظ |
| `لطیفه` | لطیفه تصادفی |
| `+سوال` | پرسش از هوش مصنوعی |
| `آمار گروه` | آمار کامل گروه |
| `قوانین` | نمایش قوانین گروه |
| `سکوت 60` | سکوت کاربر (با ریپلای) |

> برای مشاهده همه دستورات، در گروه `راهنما` را ارسال کنید.

## ساختار دیتابیس

از SQLite استفاده می‌شود. جداول اصلی:
- `user_levels` – سطح و تجربه کاربران
- `user_badges` – نشان‌های کاربران
- `custom_commands` – دستورات سفارشی
- `group_warns` – اخطارهای کاربران
- و ده‌ها جدول دیگر برای مدیریت کامل
